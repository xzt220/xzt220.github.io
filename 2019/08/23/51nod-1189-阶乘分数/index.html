1189 阶乘分数

1.0 秒 / 131,072.0 KB / 80 分 / [5级题]

Statement

1/N! = 1/X + 1/Y（0<x<=y），给出N，求满足条件的整数解的数量。例如：N = 2，1/2 = 1/3 + 1/6，1/2 = 1/4 + 1/4。由于数量可能很大，输出Mod 10^9 + 7。

输入

输入一个数N（1 <= N <= 1000000)。

输出

输出解的数量Mod 10^9 + 7。

输入样例

2

输出样例

2

Solution

推推式子

把$N! - X$看成一个整体，那么题目就变成了求$(N!)^2$的约数个数了
因为要满足$X \leq Y$，那么答案就是约数个数除以2上取整
